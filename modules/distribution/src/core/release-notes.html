Release Note - WSO2 IoT Server Version 3.3.0

We are pleased to announce WSO2 IoT Server 3.3.0.

WSO2 IoT Server is one of the most adaptive Apache licensed open source IoT platforms available today. It provides best of breed technologies for device manufacturers to develop connected products as well as rich integration and smart analytics capabilities for system integrators to adopt devices into systems they build.

These capabilities involve device management, smart analytics, API and app management for devices, transport extensions for MQTT, XMPP and many more.

What's new in WSO2 IoTS 3.3.0

  • Support for DEP(Apple Device Enrolment Program)
  • GDPR compliance for privacy requirements
  • Remote Control and screen mirroring feature for android
  • Ability to enforce geolocation bound policies
  • Ability to enforce operations/policies on devices based on input from Analytics/CEP engine
  • Ability to introduce seperate agents (iOS/Android) for each tenant
  • Process notification responses from device with analytics
  • Siddhi extensions for device management
  • MQTT Topic Browser
  • Ability to configure IoTS with a federated IDP (OpenID Connect)
  • Improvements to notification management (Manage/Clear notifications)
  • Scripts to change admin credentials and change IPs
  • Improvements to Android/iOS agents
  • Overall improvements to UI, functionality, speed, stability and Bug fixes
